Reflow Oven Controller - Quick Reference

Button Controls

B Short Press Main screen: Open menu
Menu: Select item
Edit value: Save & return
Confirm screen: Start operation
Error screen: Clear error
B Hold 2 sec Go back one menu level
B Hold 5 sec When idle: Start operation (shows confirm)
When running: STOP operation

Encoder Controls

E Rotate Menu: Navigate up/down
Edit value: Adjust value (+/- 1)
Running: Adjust target temp
E Rotate Fast Accelerated adjustment (+/- 5)

Filament Drying Presets

PLA: 45°C / 4 hrs PETG: 65°C / 4 hrs ABS: 70°C / 4 hrs Nylon: 80°C / 6 hrs TPU: 50°C / 4 hrs

Reflow Profile (Leaded Sn63/Pb37)

Phase Temp Duration
Preheat to 130°C -
Soak 130-160°C 90 sec
Reflow 210°C peak 45 sec
Cooling to 80°C -

Menu Structure

  • Select Mode
    • Reflow Solder - PCB soldering
    • Filament Drying - 3D printer filament
  • Reflow Settings
    • Peak Temp - 195-230°C
    • Soak Temp - 120-170°C
    • Soak Time - 60-120 sec
  • Filament Settings
    • Filament Type - Preset selection
    • Temperature - 40-100°C
    • Duration - 1-24 hours
  • Network Settings
    • WiFi Status
    • MQTT Status
  • System Info - FW version, IP, etc.

Display Status Indicators

Indicator Meaning
WiFi: OK Connected to network
WiFi: AP Access Point mode (setup)
WiFi: -- Not connected
MQTT: OK Connected to broker
MQTT: .. Connecting...
MQTT: -- Not connected

Web Interface

Dashboard: http://[device-ip]/
Filament Settings: http://[device-ip]/filament
Network Setup: http://[device-ip]/setup

Safety: Start can only be done via physical button (hold 5 sec). Stop is available via web/MQTT.
